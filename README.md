# 🚀 Jornada DEV — TOTVS Paulista

**👨‍💻 Desenvolvedor:** Gabriel Colpani

---

## 🎯 Foco Principal
Solidificar a **lógica de programação em Harbour** como etapa fundamental e preparatória para o desenvolvimento em **ADVPL** focado no ERP **Protheus**.

---

## 📚 Objetivos de Aprendizado
O objetivo principal deste repositório é documentar a minha evolução técnica e consolidar na prática os conhecimentos em:

* Lógica de programação
* Desenvolvimento de sistemas
* Protheus
* ADVPL
* Orientação a objetos
* Desenvolvimento de projetos completos

---

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** Harbour 3.0 (Família xBase)
* **Editor de Código:** Visual Studio Code (VS Code)
* **Compilador:** `hbmk2`
* **Sistema Operacional Base:** Windows

---
