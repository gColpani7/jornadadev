# Exercício 2 — Pseudocódigo

## a. Calcular a área de um retângulo (base × altura)

**Algoritmo:**

1. Início.
2. Ler o valor da base.
3. Ler o valor da altura.
4. Calcular a área: `área = base × altura`.
5. Exibir o valor da área.
6. Fim.

---

## b. Verificar se um número é par ou ímpar

**Algoritmo:**

1. Início.
2. Ler um número.
3. Verificar se o resto da divisão do número por 2 é igual a 0.
4. Se for igual a 0, exibir **"O número é par"**.
5. Caso contrário, exibir **"O número é ímpar"**.
6. Fim.

---

## c. Encontrar o maior entre três números

**Algoritmo:**

1. Início.
2. Ler o primeiro número.
3. Ler o segundo número.
4. Ler o terceiro número.
5. Comparar os três números.
6. Identificar qual possui o maior valor.
7. Exibir o maior número.
8. Fim.
