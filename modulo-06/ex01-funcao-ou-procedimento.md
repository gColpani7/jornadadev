
Exercício 1 — Função ou procedimento? (conceitual)

Para cada item abaixo, responda se é uma função (retorna um valor) ou um procedimento (só executa uma ação) e justifique em uma frase:


Str() — é uma função, pois convente e retorna uma string
QOut() — é um procedimento (não retorna nada)
Date() — é uma função, retorna uma string com a data atual
Len() — é uma função, retorna um inteiro INT
Upper() — é uma função, converte uma string para maiuscula e depois retorna uma nova já formatada
ClearScreen() — é um procedimento, não retorna nada, apenas limpa o terminal.
